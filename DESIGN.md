# Personal Website Design Template

## 1. Brand Identity

### Color Palette
- **Primary Color**: `#0369A1` - Main brand color (Tech Blue)
- **Secondary Color**: `#0EA5E9` - Accent color (Tech Blue)
- **Background**: `#F1F5F9` - Page background (Tech Blue)
- **Text Color**: `#0F172A` - Main text (Tech Blue)
- **Accent/Highlight**: `#06B6D4` - Links, buttons, highlights (Tech Blue)

### Typography
- **Headings**: `Georgia` — 600 (serif)
- **Body**: system sans (`-apple-system`, `Segoe UI`, `Roboto`) — 400
- **Code**: `Menlo`

### Logo/Branding

- Logo style: Wordmark in Georgia (600) with a small underline/dot accent in #06B6D4
- Favicon: Simplified wordmark dot or period in #0369A1 (export favicon.ico, favicon-32x32.png)
- Theme: Light base #F1F5F9 with Tech Blue accents (--primary: #0369A1, --accent: #06B6D4)
## 2. Layout Structure

### Header

- Navigation menu (Home, About, Projects, Blog, Contact)
- Logo/Name placement
- Mobile menu icon
- Theme toggle 

### Hero Section

- Brief introduction/tagline
- Call-to-action button
- Profile image/avatar
- Social media links

### Main Sections

#### About Me

- Professional summary
- Skills and expertise
- Technologies/Tools
- Downloadable resume/CV

#### Projects/Portfolio

- Project cards with:
  - Thumbnail image
  - Project title
  - Brief description
  - Technologies used
  - Links (live demo, GitHub)
- Filter/category options

#### Blog (optional)

- Recent posts
- Categories/tags
- Search functionality
- Read more links

#### Contact

- Contact form (name, email, message)
- Email address
- Social media links
- Location (optional)

### Footer

- Copyright notice
- Social media icons
- Quick links
- Back to top button

## 3. Design Elements

### Spacing

- Section padding: 1.5rem (24px)
- Container max-width: 1100px
- Element margins: 0.75rem (12px)

### Components

- **Buttons**: Style, hover effects, padding
- **Cards**: Border radius, shadow, hover effects
- **Forms**: Input styling, validation states
- **Images**: Border radius, aspect ratio, hover effects

### Animations/Transitions

- Page load animations
- Scroll effects
- Hover states
- Transition duration: 0.3s

## 4. Responsive Design

### Breakpoints

- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px

### Mobile Considerations

- Hamburger menu
- Stack sections vertically
- Touch-friendly button sizes
- Optimized images
